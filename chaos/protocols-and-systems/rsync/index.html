Rsync

Main points

  • Copy files locally and remotely.
  • Has a delta-transfer algorithm: sends only the differences between two files if the destination already has a copy.
  • Used for backups and mirroring.
  • Uses port TCP 873.
  • Can use SSH for secure file transfers.

Commands

# Scan for rsync with nmap:
nmap -sV -p 873 <IP address>

# Probe for accessible shares:
nc -nv <IP address> 873 # Enter "list" once connected.

# Enumerate an open share:
rsync -av --list-only rsync://127.0.0.1/<share name> # Remove --list-only to sync all files.

# Sync all files over SSH:
rsync -av -e "ssh -p<port>" rsync://127.0.0.1/<share name>
