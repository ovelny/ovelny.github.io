WMI

Main points

  • Windows Management Instrumentation.
  • Microsoft's implementation and extension of the Common Information Model (CIM).
  • Allows read and write access to nearly all settings on windows systems.
  • Can be accessed with powershell, vbscript or the Windows Management Instrumentation Console (WMIC).
  • Has several programs and databases / repositories.
  • Uses port TCP 135 for initializing the communication, but then moves the exchange to a random port.

Commands

# Connect with wmiexec.py:
wmiexec <username>:"<password>"@<IP address> "<hostname>"
