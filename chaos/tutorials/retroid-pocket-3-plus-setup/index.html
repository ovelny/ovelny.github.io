Retroid pocket 3+ setup

This document aims to be a full guide to configure the Retroid Pocket 3+. Unfortunately, it's a mess to set everything up and guides often contradict themselves depending on personal preferences. Here, I lay out mine so I can always reproduce my setup.

Initial setup

In the initial setup wizard:

  • Pick English.
  • Connect to Wi-Fi.
  • Disable google play services.
  • Select all preinstalled apps.
  • Pick RP3+ launcher.

Android settings

  • Go to settings -> display -> advanced, set screen timeout to 30 min (maximum).
  • Go to settings -> handheld settings -> input -> controller style, select xbox.
  • Go to settings -> handheld settings -> input -> disable prevent press home button accidentally.
  • Go to settings -> security -> configure screen lock.
  • Go to settings -> system -> languages and input -> gboard and set keyboard layout.

In top menu (drag it down), disable floating icon so it doesn't appear while playing fullscreen.

Download f-droid and install apps

Install f-droid with the default browser, allow permissions to run the .apk, then run f-droid and install the following apps:

  • Aurora store.
  • Amaze.

Launch aurora store, set default settings during the wizard and download daijisho with it. We'll configure it later in this guide.

Set up SD card

  • Insert empty SD card, tap notification to set up: select use for portable storage.
  • In RP3+ launcher -> emulation -> setup -> create ROM folders, select default SD card directory.
  • Plug your SD card somewhere else and copy your BIOSes and ROMs in RetroidPocketGames directories.

Set up emulation systems and ROMs

  • In RP3+ launcher -> emulation -> systems, select emulation systems you want to use.
  • Tap all select platform game -> roms -> add -> select default SD card Directory.
  • Tap scan.

All your games should show up. If not, check if they're in the right supported format for the Retroid Pocket 3+ (see resources at the end of this document).

Enable standalone emulators

Some standalone emulators seem to run better than those found in retroarch:

  • In RP3+ launcher -> emulation -> setup -> standalone setup, pick the following emulators:
    • Mupen 64.
    • PPSSPP.
    • Duckstation.
    • AetherSX2.

For reference, here is the full list of recommended standalone emulators to pick:

  • N64:
    • M64plus FZ (same as Mupen 64).
  • Sega saturn:
    • yaba sanshiro 2.
  • Playstation:
    • duckstation.
  • Nintendo DS:
    • drastic (paid app on play store).
  • Nintendo 3DS:
    • citra MMJ.
  • PSP:
    • PPSSPP.
  • Dreamcast:
    • redream.
    • flycast (for some widescreen games).
  • Gamecube:
    • dolphin MMJR.
  • Playstation 2:
    • aetherSX2.
  • Wii:
    • dolphin MMJR.
    • dolphin MMJR2 (some games work better with this one).

Install retroarch cores

Cores in retroarch are essentially embedded emulators. They're good for most systems, even if the previous standalone emulators seem to perform better.

To install cores in retroarch:

  • Tap retroarch(AArch64) app.
  • Allow storage access and let it initialize.
  • Go to online updater -> core downloader.
  • To install a core, tap it.

Here are the recommended cores for each system:

  • Game boy / game boy color:
    • gambatte.
  • Game boy advance:
    • mGBA.
  • NES:
    • mesen.
  • SNES:
    • SNES9x current.
    • BSNES HD Beta (for games running in widescreen).
  • Sega master system / game gear:
    • genesis plus GX.
  • Sega genesis / sega CD:
    • genesis plus GX.
    • genesis plus GX wide (widescreen).
  • Sega 32x:
    • picodrive.
  • Turbografx-16 / turbofrafx-CD:
    • beetle PCE fast.
  • Arcade / neo geo:
    • final burn neo.

Fix retroarch controls

Most controls for retroarch are good by default but some buttons are missing:

  • Go to settings -> input -> scroll down and select port 1 controls:
    • Remap select button.
    • Remap L2 and R2 buttons.

Set up retroarch hotkeys

Go to settings -> input -> hotkeys:

  • Turn off confirm quit.
  • Set hotkey enable to L3 (left analog stick "click").
  • Set load state to L1.
  • Set rewind to L2.
  • Set save state to R1.
  • Set fast forward (hold) to R2.
  • Set show fps to Y.
  • Set menu toggle to X.
  • Set reset game to B.
  • Set quit retroarch to R3 (right analog stick "click").
  • Set run ahead toggle to d-pad Up.

All those keys are enabled in combination with L3 + your button, as an alternative mapping for retroarch controls.

Other retroarch settings

  • In settings -> saving, enable auto save state and load state automatically.
  • In settings -> frame throttle, set fast-forward rate to 2x.
  • In settings -> on-screen display -> on-screen notifications, disable on-screen notifications.
  • In settings -> on-screen display -> on-screen overlay, disable display overlay.

Console settings and controls

Each console has specific settings that work well with the retroid, and can also provide more faithful graphics.

I'm only listing the consoles I'm interested in at the moment. This section will be updated when I start playing on other systems.

For retroarch controls, numbers are mapped to the following buttons:

  • B -> 96
  • A -> 97
  • Y -> 99
  • X -> 100
  • L1 -> 102
  • R1 -> 103
  • L2 -> 104
  • R2 -> 105
  • select -> 109
  • start -> 108

Gameboy / gameboy color - gambatte

Start any gameboy game, and press R3 + X to access retroarch menu, then navigate to the quick menu -> shaders section:

  • Enable video shaders.
  • Go to load.
  • Pick shaders_glsl -> handheld -> lcd3x.glslp.
  • Go back to shaders section.
  • Go to shader parameters and set brighten LCD to 8.00.
  • Go back to shaders, go to save -> save core preset.

To enable original gameboy color palette, go to retroarch menu again, then:

  • Go to options -> core options.
  • Go to GB colorization, select internal.
  • Go back and select internal palette and pick Special 1.
  • Go back to internal palette menu and set interframe blending to LCD ghosting (accurate).

To set up controls:

  • Go to quick menu -> controls.
  • Go to port 1 controls.
  • Directional controls are already good.
  • Set 97 to start.
  • Set 96 to A.
  • Set 100 to select.
  • Set 99 to B.
  • Disable everything else.
  • Go back and tap save core remap file.

SNES - BSNES HD Beta

Go to your SNES games -> edit -> pick Retroarch (bsnes) in top dropdown menu (not very noticeable) -> rename retroarch core to bsnes_hd_beta_libretro_android.so -> tap save. For some reason the core is not automatically detected in RP3+ launcher.

With this core however, controls are already mapped correctly!

To set up graphics and widescreen hacks:

  • Launch any games, go to retroid quick menu -> options.
  • Enable blur emulation.
  • Enable hotfixes.
  • Set hd mode 7 - scale to 240p (1x).
  • Set hd mode 7 - perspective correction to ON (wide).
  • Set widescreen mode to enable for all scenes.
  • Keep all widescreen - background layer # options to defaults.
  • Set widescreen - sprites to render when in safe area.
  • Set widescreen - area background color to always fill.
  • Set widescreen - ignore window effects to OFF.

Besides widescreen hacks, graphics look pretty good by default.

PS1 - duckstation

Launch duckstation, enable storage permissions and tap menu -> controller settings:

  • Enable auto-hide touchscreen controller.
  • Go to port 1 tab, select controller type: analog controller (dualshock).
  • In the same tab, enable use analog sticks for d-pad in digital mode.
  • Click on each button and set them as you would expect.

Go back to main menu -> app settings -> enhancements:

  • Set resolution scale to 2x.
  • Enable PGXP geometry correction (may impact performance).

Go to main menu -> app settings -> display:

  • Set aspect ratio to 16:9.

Go back to main menu -> app settings -> ehhancements:

  • Enable widescreen hack.

Duckstation is awful to import the bios file. Nothing works except the following:

  • Go to main menu -> start file.
  • Pick a random ROM to start.
  • When it prompts you to select a BIOS, navigate to your SD card and pick scph1001.bin.

N64 - mupen64

  • Click on left menu bar -> profiles -> emulation.
  • Select rice-accurate and copy.
  • Give it a unique name when prompted.
  • Set video plugin to gliden64 (gliden64 beta).
  • Enable widescreen hack.
  • Go back to main menu -> profiles -> select profiles -> emulation profile.
  • Pick your custom profile.
  • Go back to main menu -> settings -> shaders.
  • Set shader scale factor to 1.
  • Tap add in shader passes.
  • Select FXAA.
  • Go back to main menu -> settings -> input -> set show in-game menu to when slide gesture is used.
  • Go to main menu -> settings -> display -> set rendered resolution to native.
  • In the same menu, set framerate to off.

To set up controls:

  • Go back to main menu -> profiles -> controller.
  • Select android gamepad and copy.
  • Give it a unique name when prompted.
  • Select the profile.
  • Everything should be mapped correctly, just map Z to R2 as well.
  • Go back to main menu -> profiles -> select profiles -> controller 1 profile.
  • Pick your custom profile.

Gameboy advance - mGBA

If RP3+ launcher doesn't detect mGBA as the default emulator, go to your GBA games -> edit -> pick mGBA in top dropdown menu (not very noticeable) -> tap save.

Launch any game, enter retroarch quick menu, and set up controls:

  • Go to quick menu -> controls.
  • Go to port 1 controls.
  • Directional controls are already good.
  • Set 97 to start.
  • Set 96 to A.
  • Set 100 to select.
  • Set 99 to B.
  • Disable everything else except 102 and 103 (L and R).
  • Go back and tap save core remap file.

Press R3 + X to access retroarch menu again, then navigate to the quick menu -> shaders section:

  • Enable video shaders.
  • Go to load.
  • Pick shaders_glsl -> handheld -> lcd3x.glslp.
  • Go back to shaders section.
  • Go to shader parameters and set brighten LCD to 8.00.
  • Go back to shaders, go to save -> save core preset.

Go back to quick menu -> options -> video:

  • Set color correction to game boy advance.
  • Set interframe blending to smart.

Set up daijisho

  • Launch daijisho, tap on download platforms, select your console and tap import.
  • Tap settings -> video and sounds -> disable make button sounds.
  • Back in platforms menu, for each platform, tap paths -> add more and select your games directory for the console.
  • For each platform, tap on the bottom-right edit icon -> select your emulator in player settings and tap save. For retroarch cores, make sure to select those with retroarch64.
  • For each platform, tap paths -> sync.
  • Tap settings -> library -> enable disable players warnings.
  • Tap settings -> appearance -> download platform wallpapers pack.
  • Pick alekfull-NX -> download pack -> confirm.
  • Go to android settings -> search for home, select default home app and pick daijisho.

If some games need an alternate emulator (especially useful for SNES games not working well with widescreen hacks):

  • Select game.
  • Tap edit item.
  • Scroll down and enable custom player.
  • Pick another emulator (like bsnes standard) and tap save.

If some box art is missing:

  • Download box art on the web.
  • Select game in daijisho.
  • Tap edit item.
  • Tap edit preview media and select box art.
  • Select your box art and save.

Fix some emulators with daijisho

Some emulators don't run with daijisho at first, here's how to fix them:

Mupen64:

  • Go to settings -> library -> inspect players.
  • Select n64 - org.mupen64.plusae.v3.fzurita.
  • Replace {file.uri} with {file.path} in player am start arguments.
  • Tap save.

Duckstation:

  • Same issue, fix the same parameter in psx - Duckstation.

Backup and restore

You've been through all of this and never want to do it again. Same here. How do we backup everything?

As far as i know, we can only make a copy of the SD card by copying its RetroidPocketGames directory. Since we're dealing with an android device, full system backups are not a thing, even with adb.

I'd like to be wrong about this however, and hope to find something that works.
