Khal usage

I keep forgetting khal commands so this document serves as a cheatsheet.

Adding an event

        day
         | month
         |  | year
         |  |  |  hour (optional)
         v  v  v     v
khal new 17/04/2024 15:20 "event description here"
khal new 17/04/2024 15:20 17:20 "event description here" # Custom end time (default: 1h).
khal new 17/04/2024 19/04/2024 "event description here" # Multi-day event.

Deleting an event

khal edit [--show-past] "search string" # Will prompt for edit / deletion options.
