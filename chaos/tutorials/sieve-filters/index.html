Sieve filters

Sieve rule to send everything in spam unless the address is already in your allowlist:

require ["fileinto", "extlists"];

# If the address is not in the allowlist,
# send to spam. That's it.
if not header :list "from" ":incomingdefaults:inbox"
{
  fileinto "Spam";
}
