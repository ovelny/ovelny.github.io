WPScan

WordPress Security Scanner.

Commands

# Enumerate all plugins.
# -t uses 500 threads:
wpscan -e ap -t 500 --url <URL>

# Enumerate wordpress users:
wpscan -e u -t 500 --url <URL>

# Try to bruteforce a single account:
wpscan --url <URL> -P <wordlist> -U <username>
