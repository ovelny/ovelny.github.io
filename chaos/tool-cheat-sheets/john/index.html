John

Identify passwords and hashes and crack them.

Tools to crack files

Many additional tools can process files and produce hashes for john to crack:

Tool Description
pd2john Convert PDF documents.
ssh2john Convert SSH private keys.
mscash2john Convert MS Cash hashes.
keychain2john Convert OS X keychain files.
rar2john Convert RAR archives.
pfx2john Convert PKCS#12 files.
truecrypt_volume2john Convert truecrypt volumes.
keepass2john Convert keepass databases.
vncpcap2john Convert VNC PCAP files.
putty2john Convert putty private keys.
zip2john Convert zip archives.
hccap2john Convert WPA/WPA2 handshake captures.
office2john Convert MS Office documents.
wpa2john Convert WPA/WPA2 hanshakes.

Commands

# Identify hash:
hashid -j <hash>

# Crack a hash:
john --format=<hash type> <hash or hash file>

# Crack with a wordlist.
# Multiple wordlists can be specified by
# separating them with a comma:
john --format=<hash type> --wordlist=<wordlist file> <hash or hash file>

# Check progress made:
john --show

# Crack with incremental mode.
# Starting from a single character, john
# will try to guess every possible combination
# of characters:
john --incremental <hash or hash file>

# Cracking files with a *2john tool:
pdf2john example.pdf > example.hash
john example.hash

# Unshadow and crack hashes in found in /etc/shadow.
# Grab /etc/passwd and /etc/shadow and run the following:
unshadow passwd.txt shadow.txt > unshadowed.txt
john --wordlist=<wordlist path> unshadowed.txt
