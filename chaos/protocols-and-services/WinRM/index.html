WinRM

Main points

  • Windows Remote Management.
  • Command line based management protocol.
  • Uses Simple Object Access Protocol (SDAP) to establish connections.
  • Must be explicitely enabled and configured with windows 10.
  • Uses port TCP 5985 (for HTTP) and 5986 (for HTTPS).
  • Uses Windows Remote Shell (WinRS) to execute commands on the remote system.
  • Enabled by default starting with windows server 2012, must be configured for older server versions and clients.

Commands

# Scan WinRM with nmap:
nmap -sV -sC <IP address> -p5985,5986 --disable-arp-ping -n

# Connect with evil-winrm:
evil-winrm -i <IP address> -u <username> -p <password>

# Execute a password attack with crackmapexec:
cme winrm <IP address> -u <username list> -p <password list>
