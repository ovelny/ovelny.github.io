Caves of Qud

A science fantasy RPG & roguelike epic. Caves of Qud is one of my favorite games and its intricate world never cease to amaze me.

Settings

The game is often updated, and there is a myriad of settings available. It's best to change them as little as possible and stick to defaults.

General settings

  • Enable fullscreen.
  • Disable allow mouse input.
  • Disable display mouse-clickable zone transition border arrows...
  • ... And only THEN, disable modern UI elements. Awful setting.
  • Set maximum auto-move squares/sec and auto-attack squares/sec to 60.
  • Enable mods.
  • Disable scripting mods.
  • Disable send anonymous gameplay statistics.

Key mapping

Press F1 to load default keys, and select HJKL mode when prompted. Change the few following keys and be done with it:

Setting Key mapping
Adventure -> Auto-explore Shift + =
Adventure -> Look M (, on AZERTY)
Adventure -> Ability Picker Q (A on AZERTY)
Status -> Open Quests A (Q on AZERTY)
Display -> Toggle sidebar Shift + \ (* on AZERTY)
